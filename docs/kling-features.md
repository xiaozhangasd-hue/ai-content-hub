# 可灵AI功能说明

## 功能概览

南都AI现已集成可灵AI（快手）的全部视频能力，包括：

| 功能 | 接口 | 说明 | 适用场景 |
|------|------|------|----------|
| **视频生成** | `/api/video` | 文生视频、图生视频 | 宣传视频、课程演示 |
| **数字人** | `/api/kling/avatar` | AI数字人视频生成 | 课程介绍、老师代言 |
| **对口型** | `/api/kling/lip-sync` | 让视频中的人物说话 | 口播视频、配音视频 |

## 功能详解

### 1. 视频生成

**接口**: `POST /api/video`

**请求示例**:
```json
{
  "prompt": "一位老师在明亮的教室里给孩子们上课，温馨的教学氛围",
  "duration": 5,
  "ratio": "16:9",
  "resolution": "720p",
  "model": "v1-5",
  "mode": "std"
}
```

**响应**:
```json
{
  "success": true,
  "taskId": "task_xxx",
  "status": "pending",
  "message": "视频生成任务已提交"
}
```

**查询进度**: `GET /api/video?taskId=task_xxx`

---

### 2. 数字人生成 🎯 新功能

**接口**: `POST /api/kling/avatar`

**功能**: 生成AI数字人说话视频

**请求示例**:
```json
{
  "name": "李老师",
  "text": "大家好，我是李老师，欢迎来到我们的少儿口才课堂！",
  "model": "v2",
  "duration": 30
}
```

**参数说明**:
- `name`: 数字人名称
- `text`: 数字人要说的文本内容
- `image`: （可选）自定义数字人形象图片URL
- `audio`: （可选）自定义音频URL
- `model`: 模型版本（v1或v2，推荐v2）
- `duration`: 视频时长（秒）

**应用场景**:
- ✅ 课程介绍视频
- ✅ 老师形象代言
- ✅ 招生宣传口播
- ✅ 课程预告

---

### 3. 对口型功能 🎯 新功能

**接口**: `POST /api/kling/lip-sync`

**功能**: 让视频中的人物说话（嘴型与音频同步）

**请求示例**:
```json
{
  "videoUrl": "https://xxx/video.mp4",
  "audioUrl": "https://xxx/audio.mp3"
}
```

**参数说明**:
- `videoUrl`: 视频URL（必须包含清晰的人物面部）
- `audioUrl`: 音频URL

**应用场景**:
- ✅ 给课程视频配音
- ✅ 制作老师口播视频
- ✅ 批量生成多语言版本

## 配置步骤

### 步骤1: 注册可灵AI账号

1. 访问 https://klingai.com/cn/dev
2. 使用手机号或快手扫码登录
3. 完成实名认证

### 步骤2: 购买资源包

1. 进入控制台 → 资源包购买
2. 选择合适的套餐（有试用额度可免费测试）
3. 完成支付

### 步骤3: 创建API Key

1. 进入控制台 → API密钥管理
2. 点击"新建API密钥"
3. 复制 **Access Key** 和 **Secret Key**（⚠️ Secret Key只显示一次）

### 步骤4: 配置环境变量

在 `.env` 文件中添加：

```env
VIDEO_PROVIDER="kling"
KLING_ACCESS_KEY="your-access-key"
KLING_SECRET_KEY="your-secret-key"
```

## 教培机构应用场景

### 场景1: 课程介绍视频

使用**数字人**功能，快速生成老师介绍课程的视频：

```javascript
// 生成课程介绍视频
await fetch('/api/kling/avatar', {
  method: 'POST',
  body: JSON.stringify({
    name: '张老师',
    text: '大家好！我是张老师，专注少儿口才培训10年。我们的课程采用小班教学，让孩子在快乐中学会自信表达！',
    model: 'v2',
    duration: 30
  })
});
```

### 场景2: 招生宣传口播

使用**视频生成**功能，创建招生宣传片：

```javascript
// 生成招生宣传视频
await fetch('/api/video', {
  method: 'POST',
  body: JSON.stringify({
    prompt: '明亮的教室里，孩子们在老师的指导下进行口才训练，充满欢声笑语，温馨的学习氛围',
    duration: 5,
    ratio: '16:9'
  })
});
```

### 场景3: 多语言课程版本

使用**对口型**功能，制作多语言版本：

1. 先用可灵生成老师形象视频
2. 准备不同语言的配音文件
3. 使用对口型功能，生成多个语言版本

## 常见问题

### Q: 视频生成需要多久？
A: 5秒视频约需2-3分钟，时长越长生成时间越长

### Q: 数字人支持自定义形象吗？
A: 支持！上传形象图片即可生成自定义数字人

### Q: 对口型对视频有什么要求？
A: 视频必须包含清晰的人物正面面部，光线充足效果更好

### Q: 如何查询任务状态？
A: 使用返回的taskId调用查询接口，建议每10秒轮询一次
